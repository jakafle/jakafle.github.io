Introduction

As a physics student, computation exists at the heart of what I do. Simulating physical processes as well as mathematical models is one forte of physical and mathematical sciences, which is both fun and worth learning. This article, I will focus on the preliminaries of scientific computing — getting started with the software to install and configure.

A) Programming Languages

While you can use any of the programming languages, especially when it comes to scientific computing, C, C++, Fortran, Python, and Julia are preferred. Moreover, it is easier to configure and run applications in UNIX based operating systems, so I'll write about configuring your Linux system for scientific computing. Since I use Ubuntu 22.04 currently, I'll address all commands to this system.

i. C, C++, and Fortran
C, C++, and Fortran are compiled languages. To install the compiler (GNU compiler collections) and the gnu debugger, you can use the following commands.
% sudo apt install gcc g++ gfortran gdb
Moreover, gnuplot is widely used for creating scientific plots and can be installed using
% sudo apt install gnuplot
You'll also need a text editor to write programs. While the default text editor in Ubuntu 22.04, gedit works fine, you might want to use vim, emacs, or Visual Studio Code. You can install these using the following commands.
% sudo apt install vim emacs
% sudo snap install code --classic

ii. Python
Python is preinstalled in ubuntu, so you only need to install the libraries that you require. numpy, scipy, astropy, pandas, sympy, and matplotlib are popular libraries that you will use frequently for scientific computing. Moreover, jupyter notebooks are popular among scientists. You can install them from the ubuntu repositories using
% sudo apt install python3-numpy python3-scipy python3-astropy python3-pandas python3-sympy python3-matplotlib ipython3 jupyter jupyter-notebook
 While you can also use pip to install the python packages, I would suggest not to use sudo pip to install them. Rather, pip install --user or a virtual environment will be a good option.
For some users, who don't have administrator access, or in general for easier installation, Anaconda distribution of python will be a good option. Besides, miniconda (a mini python version, which does not have all the scientific libraries in Anaconda), or mambaforge (uses mamba as a package manager instead of conda) will be better if you want to install the packages yourself. Download the python_installer.sh (note that this is a dummy script name, you can go to the respective downloads page to download the scripts) script and open terminal and run the following:
% bash python_installer.sh
Agree through the terms and conditions, and change (or keep default) the installation directories to install the Anaconda/miniconda/mambaforge python distributions. Then, you can install the packages as using conda or mamba. Since I use mambaforge, I'll write the command using mamba.
% mamba install numpy scipy astropy pandas sympy matplotlib ipython jupyter notebook jupyterlab

iii. Julia
Julia is availabe as a snap on Ubuntu. However, I recommend installing the generic binaries for your system using the Julia downloads page. This method also works if you want multiple versions of Julia installed. The installation instructions are in the Julia downloads page. To install packages, you can use the commands below:

B) Other Software

Besides the above programming languages, other languages and softwares are essential to the workflow when doing scientific computing. One of the important tools is LaTeX, which is widely used in academia for publishing scientific documents. You will need a LaTeX distribution, and Ubuntu has texlive in its repositories. Along with, you need a LaTeX editor, I install both TeXstudio and Setzer, and use whichever suitable. LyX is a document processor, which does not need any experience with LaTeX, thus can be helpful if you want formatted documents without any hassle of using LaTeX. Pandoc will also be useful if you want to convert from one document format to another. You can install these tools using the command below.
% sudo apt install texlive-full texstudio lyx pandoc

For graphics and making diagrams, gimp and inkscape will come handy. You can install them with the command
% sudo apt install gimp inkscape

For astronomy enthusiasts, Kstars, Stellarium, Siril, and DS9 are available in the repositories. Qalculate! is a feature rich calculator for your system, and I definitely recommend to install it. If you don't want to write codes and simply want to analyze data, LabPlot by KDE can be used. Other scientific applications like GNU Octave, Scilab, Paraview, Quantum Espresso, GNU Data Language, IRAF, and R can also be installed in the system using similar commands or the binaries provided by the software managers.

This article went through a brief overview of softwares used in science. Next part, we'll go into details of some of the tools mentioned here. Happy computing!
